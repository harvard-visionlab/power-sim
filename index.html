Get started with the Power-Sim Toolbox.

The toolbox is a collection of MATLAB functions for simulating experimental designs using bootstrap resampling, a simulation technique.

Download the toolbox using one of the above links, then read the paper or the tutorial.

If you wish to acknowledge use of this tool or technique when you publish your research, you might write something like,

The power of experimental design was calculated using bootstrap resampling or pilot data, using the Power-Sim Toolbox (Strong & Alvarez, XXX; memtoolbox.org).
