Get started with the Power-Sim Toolbox.

The toolbox is a collection of MATLAB functions for calculating the power of experimental designs using bootstrap resampling, a technique that uses pilot data to simulate running experimental designs many times.

Currently, the toolbox allows simulation of 1-factor (t-tests) and 2-factor (ANOVAs) designs, allowing researchers to calculate how changing sample size, number of trials per condition, and participant-exclusion criteria will influence their statistical power.

Download the toolbox using one of the above links, then read the paper or the tutorial.
